# Build OS Portfolio

Static portfolio site inspired by premium product pages, rebuilt as an original developer portfolio for `Jivl1`.

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173` from this folder.

## Deploy

This site is static and ready for Vercel. The root directory should be `portfolio-os`, with no build command and output directory set to `.`.
